# Yuri's React/Redux/Immutable/Webpack Boilerplate

## Installation

```
git clone git@github.com:ninjakttty/react-boilerplate.git
npm install
```

## NPM Scripts

* start -- start in dev mode
* dev -- same as above
* build -- build production dist folder
* test -- run tests
* eject -- eject from react boilerplate
